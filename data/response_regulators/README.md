# Input data

The following files are downloaded from FIXME on first execution:

* `PF00072.full.sto`, `PF00196.full.sto`, `PF00486.full.sto`, `PF04397.full.sto`: Full Pfam alignments in Stockholm format (Pfam 37.4).
* `pdb_rr.fasta`: Representative PDB sequences for each response regulator subfamily (PF00486: 1NXS, PF04397: 4CBV, PF00196: 4E7P).

Other files that appear in this directory are derived from these inputs.
